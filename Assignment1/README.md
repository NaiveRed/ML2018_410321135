# Image Decryption using Linear Percetpron

## 說明

根據老師所給的圖片資料(I, E, key1, key2)，透過 Linear Percetpron 找出加密的權重。  
最後能透過該組權重來對圖片進行加密或解碼。

## 環境配置

使用 python 3 (3.6.5)

相關套件請參考 [requirements.txt](requirements.txt)

## 檔案

I: 原始圖片  
E: 加密後的圖片  
key1, key2: 加密鑰匙
Eprime: 待解碼的圖片  
(以上檔案為老師所提供)

weights.txt: 所找出的權重  
weights.dat: 所找出的權重(ndarray)
